# 4.  STECKVERBINDER - VG 95319

## 4. 1 Übersicht VG 95319

Die Normenreihe **VG 95319** definiert die Anforderungen an elektrische Steckverbinder und Steckvorrichtungen für militärische Anwendungen. Sie ist das deutsche Äquivalent zur US-amerikanischen MIL-DTL-38999 Serie.

### 4.1.1 Struktur der Normenreihe

| Teil | Bezeichnung | Beschreibung |
|------|-------------|--------------|
| **VG 95319-1** | Übersichtsnorm | Allgemeine Anforderungen, Klassifizierung |
| **VG 95319-100** | Fachgrundnorm | Prüfverfahren und Qualitätsanforderungen |
| **VG 95319-102** | Übersicht Detail | Detailspezifikationen für Bauarten |
| **VG 95319-1006** | Rundsteckverbinder | Gewinde-Kupplung (Screw Coupling) |
| **VG 95319-1007** | Schnellkupplung | Bajonett-Kupplung (Bayonet) |
| **VG 95319-1009** | Kontakte | Stift- und Buchsenkontakte |
| **VG 95319-1010** | Gehäuse Überwurf | Jam-Nut Gehäuse |
| **VG 95319-1011** | Endgehäuse | Backshells, Zugentlastungen |
| **VG 95319-1012** | Gehäuse Flansch | Flansch-Gehäuse (Square Flange) |

**Quelle:** DIN Media VG 95319-1, GlobalSpec Standards Database

### 4. 1.2 Kompatibilität mit MIL-SPEC

| VG-Norm | MIL-SPEC Äquivalent | Anwendung |
|---------|---------------------|-----------|
| VG 95319-1006 | MIL-DTL-38999 Serie III | Rundsteckverbinder mit Gewinde |
| VG 95319-1007 | MIL-DTL-38999 Serie I/II | Schnellkupplung (Bajonett) |
| VG 95319-1009 | MIL-PRF-29504 | Kontakte (Crimp/Löt) |
| VG 95319-1011 | MIL-DTL-38999 Backshells | Endgehäuse und Zugentlastung |

**Quelle:** Amphenol-Air LB VG95319 Products, TE Connectivity VG-Standards

## 4. 9 Montage und Verarbeitung

### 4.9.1 Crimpverfahren

```
CRIMP-PROZESS FÜR VG 95319-1009 KONTAKTE
════════════════════════════════════════

SCHRITT 1: VORBEREITUNG
───────────────────────
☐ Korrekte Leitung wählen (Querschnitt prüfen)
☐ Abisolierlänge nach Herstellervorgabe
☐ Korrekten Kontakt wählen (Größe, Typ)
☐ Kalibriertes Crimpwerkzeug verwenden

SCHRITT 2: ABISOLIEREN
──────────────────────
☐ Isolierung sauber entfernen (kein Schnitt in Litze!)
☐ Abisolierlänge: typisch 3,0 - 4,0 mm (kontaktabhängig)
☐ Litzenenden nicht verzinnen (nur Crimp!)
☐ Litzenenden nicht verdrillen

SCHRITT 3: CRIMPEN
──────────────────
☐ Leitung in Kontakt einführen (bis Anschlag)
☐ Litzenenden sichtbar im Kontrollfenster
☐ Crimp ausführen (korrekter Locator)
☐ Isolationscrimp ausführen (falls vorhanden)

SCHRITT 4: KONTROLLE
────────────────────
☐ Optische Kontrolle (Bellmouth, keine Risse)
☐ Zugprüfung (Haltekraft nach Tabelle)
☐ Elektrische Prüfung (Widerstand < 10 mΩ)
```

### 4. 9.3 Kontaktbestückung

```
KONTAKT-EINBAU IN STECKVERBINDER
════════════════════════════════

WERKZEUGE:
☐ Einbauwerkzeug (Insertion Tool) - z.B. DMC DRK-16
☐ Ausbauwerkzeug (Extraction Tool) - z.B. DMC DRK-16

EINBAU-PROZESS:
1. Steckverbinder fixieren
2.  Korrekte Kavität identifizieren (nach Zeichnung)
3.  Kontakt in Einbauwerkzeug setzen
4. In Kavität einführen bis "Klick" hörbar
5.  Leicht ziehen → Haltekraft prüfen

AUSBAU-PROZESS:
1.  Ausbauwerkzeug in Kavität einführen
2.  Haltefeder lösen
3.  Kontakt herausziehen
4.  Kontakt auf Beschädigung prüfen

⚠️ NIEMALS ohne Werkzeug arbeiten! 
   Beschädigte Kontakte = AUSSCHUSS
```

### 4.9.4 Häufige Fehler

| Fehler | Ursache | Abhilfe |
|--------|---------|---------|
| Kontakt sitzt nicht fest | Falsche Kavität, beschädigte Feder | Kontakt austauschen |
| Crimp zu locker | Falsches Werkzeug, falsche Einstellung | Neu crimpen |
| Crimp zu fest | Übercrimpt | Kontakt ersetzen |
| Litzenbruch | Isolierung zu tief geschnitten | Neu abisolieren |
| Hoher Widerstand | Oxidation, schlechter Crimp | Kontakt ersetzen |
